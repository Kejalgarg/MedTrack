# MedTrack
“MedTrack is a project where i designed a simple medication tracking and reminder system to help users manage their medicine schedules more effectively. 

The problem we focused on was that many people forget to take medicines on time, especially when they have multiple prescriptions with different frequencies. This leads to poor medication adherence and can affect treatment outcomes. 

To address this, we created a simple interface where users can enter the medication name, dosage, frequency, and time. Once the data is entered, the system records the schedule and displays upcoming reminders so the user has a clear view of their medication plan. 

From an operational perspective, the goal was to create a structured tracking system that reduces dependency on memory and improves compliance by providing visibility of upcoming tasks. 

The interface was built using HTML and CSS for the layout and JavaScript to capture the input data and dynamically update the reminder list.”
