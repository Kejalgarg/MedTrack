# MedTrack
MedTrack – Medication Reminder System
Tagline

A simple and intuitive web-based system to help users manage and track their medication schedules effectively.

Project Description

MedTrack is a lightweight medication reminder application designed to help users keep track of their daily medicines and avoid missing important doses. The system allows users to enter medication details such as the medicine name, dosage, intake frequency, and time.

Once the details are submitted, MedTrack dynamically generates a list of upcoming reminders so that users can easily monitor their medication schedule. The application provides a clean and user-friendly interface, making it simple for users to organize and view their medication routine.

This project demonstrates how a simple digital solution can improve medication adherence and help individuals manage their health more effectively.

Problem Statement

Many people forget to take their medicines on time due to busy schedules or lack of proper tracking tools. Missing medication doses can lead to serious health complications.

MedTrack addresses this issue by providing a simple platform where users can schedule and monitor their medication reminders in one place.

How It Works

The user enters medication details including:

Medication name

Dosage

Frequency

Reminder time

When the user clicks Set Reminder, the system:

Collects the input data

Creates a reminder entry

Displays it in the Upcoming Reminders list

The reminder list dynamically updates each time a new medication is added.
